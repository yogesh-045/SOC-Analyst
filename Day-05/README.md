# 📚 What I Learned
top displays the most common values in a field.

rare displays the least common values.

limit controls how many results are returned.

sort arranges results in ascending or descending order.

These commands help quickly identify suspicious users, hosts, and IP addresses.

They are commonly used in SOC investigations for anomaly detection and threat hunting.
