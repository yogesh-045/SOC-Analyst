# 📚 What I Learned
A field is an extracted piece of information from a log event.

The pipe (|) operator passes the output of one command to the next.

The table command displays only selected fields.

The fields command keeps or removes fields from the results.

The head command shows the first N events.

The tail command shows the last N events.

The sort command arranges results in ascending or descending order.

Multiple SPL commands can be combined to create efficient searches.

These commands help SOC analysts quickly organize and investigate log data.
